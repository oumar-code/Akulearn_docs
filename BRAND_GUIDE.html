Brand Guidelines

✓ DO

  • Use high contrast for accessibility
  • Animate Aku on key moments (login, level-up)
  • Maintain 2:1 clear space around logos
  • Use full color on clean backgrounds
  • Tell the story behind each logo
  • Test color contrast (WCAG AAA)

✗ DON'T

  • Rotate/distort Aku outside animation
  • Use logos smaller than 100px width
  • Place on complex/patterned backgrounds
  • Change colors arbitrarily
  • Hide the turban story from users
  • Use low-contrast color combinations

Implementation Checklist

Get Akulearn Branded

  • Add Aku logo to homepage header
  • Add rotation animation on platform load
  • Add Telhone logo to discussion/chat features
  • Add Versatech attribution to footer
  • Add founder bio to About page
  • Use brand colors in UI components
  • Test contrast ratios across all pages
  • Create brand guide for developers
  • Document logo animation specs
  • Test on mobile and desktop views
