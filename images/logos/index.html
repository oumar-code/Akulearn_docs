    Brand Logos — Asset Index¶

    This directory contains all official brand logos for the Aku Platform ecosystem. Each logo is provided as an SVG vector file for maximum scalability and fidelity across all use-cases.

    Directory Structure¶

    docs/images/logos/
├── aku-brand-horns-up.svg      — Aku Platform primary brand logo (horns facing up)
├── aku-brand-horns-down.svg    — Aku Platform alternate brand logo (horns facing down)
├── telhone-logo.svg            — Telhone telecom brand logo (used by Aku eSIM service)
└── akudemy-logo.svg            — Akudemy education brand logo (used by Aku Learn service)

    Logo Descriptions¶

    Aku Brand — Horns Up (aku-brand-horns-up.svg)¶

    The primary Aku Platform brand mark. Features the Aku mascot with horns pointing upward, conveying growth, ambition, and forward momentum. Used on the main platform homepage, official marketing materials, and the Aku IG-Hub interface.

    Palette: Deep navy (#0a1628) background, amber/gold (#e8a020) mascot, white wordmark.

    Aku Brand — Horns Down (aku-brand-horns-down.svg)¶

    The alternate Aku Platform brand mark. Features the Aku mascot with horns pointing downward, representing grounding, stability, and local roots. Used for community-facing materials, the Edge Hub interface, and print collateral where the primary logo is already prominent.

    Palette: Deep navy (#0a1628) background, amber/gold (#e8a020) mascot, white wordmark.

    Telhone (telhone-logo.svg)¶

    The brand identity for Telhone, the Aku Platform's embedded telecom and eSIM service. Features a phone handset with radio signal arcs and a modern blue-on-dark palette. Used in: - Aku eSIM service documentation and UI - MVNO/eSIM provisioning portals - Device onboarding materials for eSIM-enabled devices

    Palette: Navy (#003366) background, bright blue (#00bfff) icon, white wordmark.

    Akudemy (akudemy-logo.svg)¶

    The brand identity for Akudemy, the Aku Platform's education delivery brand. Features a graduation cap above an open book. Used in: - Aku Learn service documentation and UI - Student-facing mobile app screens - Educator and school admin portals - Community learning programme materials

    Palette: Deep indigo (#1a237e) background, gold (#ffd600) graduation cap, white book, gold wordmark.

    Usage Guidelines¶

    Logo Primary Service Secondary Applications
    aku-brand-horns-up.svg Aku Platform (IG-Hub, Super Hub) Marketing, website hero, official documents
    aku-brand-horns-down.svg Aku Edge Hub community interface Print, community materials
    telhone-logo.svg Aku eSIM / Telhone MVNO eSIM provisioning UI, device packaging
    akudemy-logo.svg Aku Learn / Akudemy Mobile app, school portals, educator toolkits

    File Naming Convention¶

    <brand>-logo[-variant].svg

    • Replace actual logo files (placeholder SVGs) with production-quality artwork from the design team before any public release.
    • Keep source files (.ai, .fig, .sketch) in the design team's asset management system; only export SVGs here.
