# Aku-Telhone Overview

Aku-Telhone is Akulearn's connectivity initiative, designed to start lean and scale in phases without blocking early deployment.

## 3-Phase Delivery Roadmap

### Phase 1 (iDICE MVP): App-First Baseline

Users download the **Aku Telhone App**.
When they come within range of a school Edge Hub, the app detects the local Wi-Fi network, logs the user in seamlessly, and routes traffic through localized cache services.

**Why this is the MVP baseline:**
- No SIM card issuance required
- No telecommunications spectrum licensing required
- No cellular tower CAPEX required
- Software-driven rollout using smartphone Wi-Fi already available on devices

### Phase 2: Physical SIM Rollout (Local Mesh + MVNO Blend)

As coverage expands beyond initial school clusters, Aku-Telhone adds branded physical SIM cards through an MVNO/national roaming partnership with existing MNOs.

**Operating model:**
- **Outside school zones:** users run on commercial cellular towers for talk/text/light data
- **Inside school zones:** heavy data is offloaded from tower traffic to Aku Edge Hub local Wi-Fi cache

**Commercial effect:**
- Lower recurring data cost through local offloading
- Broader continuity of service when users travel outside hub zones

### Phase 3: eSIM Evolution (Borderless Expansion)

Aku-Telhone transitions from physical SIM logistics to eSIM provisioning.
Users scan a QR code inside the app and instantly download the network profile.

**Strategic fit:**
- Removes SIM manufacturing/distribution friction
- Supports multi-country growth under Coo-Cah Technologies Holdings
- Enables cross-border user continuity across Nigeria, Rwanda, Kenya, and other partner markets
