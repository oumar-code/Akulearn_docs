# Aku-Telhone Overview

Aku-Telhone is the telecommunications arm of the Akulearn platform, providing affordable mobile connectivity to learners across Nigeria through MVNO (Mobile Virtual Network Operator) services and eSIM provisioning.

## What is Aku-Telhone?

Aku-Telhone bridges the connectivity gap for underserved communities by offering:

- **Ultra-affordable data plans** tailored for educational content consumption
- **eSIM provisioning** for compatible devices, enabling instant activation
- **Physical SIM distribution** through community agents and school networks
- **MVNO infrastructure** built on top of licensed Nigerian mobile network operators

## Key Services

| Service | Description |
|---------|-------------|
| **Telhone App** | Mobile application for SIM management, data top-ups, and account control |
| **eSIM** | Remote SIM provisioning for eSIM-capable smartphones and tablets |
| **Physical SIM** | Traditional SIM cards distributed via agent networks |
| **MVNO Backend** | Core network infrastructure for subscriber management and billing |

## Integration with Akulearn

Aku-Telhone is tightly integrated with the broader Akulearn platform:

- Zero-rated access to Akulearn educational content for Telhone subscribers
- Bundled data packages with Akudemy courses and AI Tutor sessions
- Single sign-on (SSO) linking Telhone and Akulearn accounts
