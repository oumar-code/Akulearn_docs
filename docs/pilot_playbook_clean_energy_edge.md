# Pilot Playbook: Clean Energy + Edge Computing in Education

## 1. Overview
This playbook guides you through planning, deploying, and evaluating a pilot project that combines clean energy and edge computing in a rural school setting using the Aku platform.

## 2. Objectives
- Provide reliable, renewable power to a school using solar/wind.
- Deploy a portable edge data center running Aku services (AI tutors, dashboards).
- Measure impact on learning outcomes, energy savings, and system uptime.

## 3. Prerequisites
- Site selected (school with limited/no grid access)
- Solar/wind hardware and portable edge server available
- Aku platform software and content ready for deployment
- Local project champion (teacher/admin)

## 4. Step-by-Step Deployment
### A. Preparation
1. Conduct site survey (power needs, space, connectivity)
2. Secure permissions and local support
3. Prepare hardware (solar panels, batteries, edge server)
4. Preload Aku platform software and content

### B. Installation
1. Install solar/wind system and connect to edge server
2. Set up local Wi-Fi for school devices
3. Deploy Aku services (AI tutor, dashboards, content)
4. Train local staff on system use and troubleshooting

### C. Operation
1. Monitor system uptime and energy usage
2. Support teachers and students in using digital tools
3. Collect feedback and usage data

### D. Evaluation
1. Assess learning outcomes (pre/post surveys, engagement metrics)
2. Analyze energy savings and uptime
3. Document lessons learned and recommendations

## 5. Troubleshooting & Support
- Common issues (power, connectivity, software)
- Contact info for technical support

## 6. Reporting & Next Steps
- Prepare a case study with outcomes and impact
- Plan for scale-up or handover to local team
