# Gusau Pilot Schedule (Gantt-style overview)

## High-level schedule (12 weeks)

- Weeks -8 to 0: Preparation
  - Stakeholder engagements, procurement orders placed, finalize school list
- Weeks 0–2: Site surveys & staging
  - Conduct surveys using `hardware/deployment/zmf_site_survey.csv`
  - Stage 10 kits, pre-load Aku Learn content
- Weeks 2–6: Installations (10 Edge Hubs)
  - Week 2: Install 3 hubs
  - Week 3: Install 3 hubs
  - Week 4: Install 2 hubs
  - Week 5: Install 2 hubs
  - Each install day: morning install, afternoon commissioning & teacher intro
- Weeks 6–10: Teacher onboarding & community launches
  - Teacher 2-day workshops at central training site
  - Community awareness events at each school
- Weeks 10–14: Stabilize & metrics collection
  - Continuous monitoring, bug-fixes, second round of content updates

## Example Week-by-week activities (more granular)
- Week -4: Procurement finalization, book logistics vehicles, order spares
- Week -2: Training materials ready, trainer selection
- Week 0: Site survey complete, staging at Gusau warehouse
- Week 2–6: Install rotation (2–3 installs/week)
- Week 6–8: Teacher workshops (2 cohorts)
- Week 8–12: Monitor & iterate

## Owners & Escalation
- Deployment Lead: onsite coordination
- Logistics: transport, warehouse
- Training Lead: teacher workshops
- Monitoring Lead: dashboards and support tickets
