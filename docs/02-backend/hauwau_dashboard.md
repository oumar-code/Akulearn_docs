# Hauwau Abubakar Dashboard

The Hauwau Abubakar Dashboard is a special-access interface designed for the Exam Prep & Access Coordinator role. It combines JAMB exam preparation tools with a broad set of platform accesses across multiple dashboards and content areas.

## Key Features

- **JAMB Exam Prep:** Full access to the JAMB preparation module, including practice questions, mock exams, past-question archives, and performance analytics.
- **WAEC & NECO Access:** Access to WAEC and NECO exam preparation content and assessments.
- **Student Dashboard View:** Ability to view and monitor the student dashboard for oversight and support.
- **Teacher Dashboard View:** Access to the teacher dashboard to coordinate curriculum and content delivery.
- **Content Management:** Create, review, and publish learning content across subjects and exam categories.
- **Analytics:** View platform-wide analytics and exam readiness metrics.
- **School Admin Dashboard View:** Access to the school admin dashboard for school-level coordination.

## Access

This dashboard is exclusively assigned to **Hauwau Abubakar** in the `Exam Prep & Access Coordinator` role. Access is governed by role-based authentication and is configured in [`team.py`](../../team.py).

## Navigation

From the main navigation, select **Backend > Hauwau Dashboard** to access this page.

For technical details, see the backend API documentation and the team configuration in `team.py`.
