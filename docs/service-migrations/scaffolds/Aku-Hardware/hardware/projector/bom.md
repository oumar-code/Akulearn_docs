## Notes

- **Projection screen** is optional if a white-painted wall is available.
- For indoor night-time classes, the ViewSonic PA503W 3600 lm unit provides excellent
  visibility at 80-inch screen size from a 2.5 m throw distance.
- For permanent installations, replace the tripod (P5) with a ceiling mount (P3).
- The presenter remote (P7) pairs with the Aku-SmartBoard facilitator interface for
  slide-advance and pointer functionality.
