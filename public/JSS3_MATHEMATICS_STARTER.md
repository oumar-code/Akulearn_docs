## 1. KEY CONCEPTS (Master These)

### Concept 1: Quadratic Expressions and Equations

**Quadratic** means "degree 2" — highest power is x².

**Standard form**: ax² + bx + c = 0

**Solving by Factorisation**:
```
x² + 5x + 6 = 0
→ Find two numbers that multiply to +6 and add to +5: (2 and 3)
→ (x + 2)(x + 3) = 0
→ x = −2  or  x = −3
```

**Solving by Completing the Square**:
```
x² + 6x − 7 = 0
x² + 6x = 7
x² + 6x + 9 = 7 + 9   (add (b/2)² = 9 to both sides)
(x + 3)² = 16
x + 3 = ±4
x = 1  or  x = −7
```

**Quadratic Formula** (use when factorisation is not obvious):
```
x = [−b ± √(b² − 4ac)] / 2a

For 2x² − 3x − 2 = 0  (a=2, b=−3, c=−2):
Discriminant = (−3)² − 4(2)(−2) = 9 + 16 = 25
x = [3 ± 5] / 4  →  x = 2  or  x = −½
```

**Discriminant (b² − 4ac)**:
- > 0 → two different real roots
- = 0 → one repeated root
- < 0 → no real roots

### Concept 2: Pythagoras' Theorem and Trigonometry (Intro)

**Pythagoras' Theorem** (right-angled triangles only):
```
c² = a² + b²    where c is the hypotenuse (longest side, opposite the right angle)

Finding hypotenuse: c = √(a² + b²)
Finding a leg:      a = √(c² − b²)
```

**Pythagorean triples** (memorise these):
```
3–4–5   | 5–12–13  | 8–15–17  | 7–24–25
(and all multiples, e.g., 6–8–10, 9–12–15)
```

**Basic Trigonometry** (SOHCAHTOA):
```
Sin θ = Opposite / Hypotenuse
Cos θ = Adjacent / Hypotenuse
Tan θ = Opposite / Adjacent
```

**Special angles** (for BECE):
| θ | Sin θ | Cos θ | Tan θ |
|---|-------|-------|-------|
| 30° | ½ | √3/2 | 1/√3 |
| 45° | √2/2 | √2/2 | 1 |
| 60° | √3/2 | ½ | √3 |

### Concept 3: Mensuration — Volumes and Surface Areas of Solids

| Solid | Volume | Total Surface Area |
|-------|--------|-------------------|
| Cuboid (l×w×h) | lwh | 2(lw + lh + wh) |
| Cube (side a) | a³ | 6a² |
| Cylinder (r, h) | πr²h | 2πr(r + h) |
| Cone (r, h, slant l) | ⅓πr²h | πr(r + l) |
| Sphere (r) | ⁴⁄₃πr³ | 4πr² |

**π = 22/7** unless told otherwise

**Example**: A cylinder has radius 7 cm and height 10 cm. Find volume and curved surface area.
```
Volume = πr²h = 22/7 × 7² × 10 = 22/7 × 49 × 10 = 1,540 cm³
Curved SA = 2πrh = 2 × 22/7 × 7 × 10 = 440 cm²
```

### Concept 4: Probability

**Basic Probability**:
```
P(event) = number of favourable outcomes / total possible outcomes

Range: 0 ≤ P ≤ 1
P = 0   → impossible
P = 1   → certain
P = 0.5 → equally likely
```

**Complement rule**: P(A does NOT happen) = 1 − P(A)

**Example**: A bag contains 5 red, 3 blue and 2 green balls. A ball is picked at random. Find:
```
Total = 5 + 3 + 2 = 10
P(red)   = 5/10 = ½
P(blue)  = 3/10
P(green) = 2/10 = 1/5
P(not red) = 1 − ½ = ½
```

**OR rule** (mutually exclusive events):
P(A or B) = P(A) + P(B)

**AND rule** (independent events):
P(A and B) = P(A) × P(B)

### Concept 5: Variation — Direct and Inverse

**Direct Variation**: y is directly proportional to x → y = kx
- If y doubles, x doubles
- k (constant of proportionality) = y/x

**Inverse Variation**: y is inversely proportional to x → y = k/x
- If y doubles, x halves
- k = xy

**Joint Variation**: y ∝ xz → y = kxz

**Example — Direct**:
- y = 12 when x = 4. Find y when x = 9.
- k = y/x = 12/4 = 3 → y = 3x → y = 3 × 9 = **27**

**Example — Inverse**:
- y = 8 when x = 5. Find x when y = 20.
- k = xy = 8 × 5 = 40 → y = 40/x → 20 = 40/x → x = **2**

## 2. SAMPLE PROBLEMS (5 Increasing Difficulty)

### Problem 1 (EASY — Pythagoras)
**Q**: A ladder 13 m long leans against a vertical wall. The foot of the ladder is 5 m from the wall. How high up the wall does the ladder reach?

**Solution**:
```
h² = 13² − 5²
h² = 169 − 25 = 144
h  = √144 = 12 m
```
**Answer**: **12 m**

---

### Problem 2 (EASY — Probability)
**Q**: A fair die is rolled. Find the probability of getting a prime number.

**Solution**:
```
Outcomes on a die: 1, 2, 3, 4, 5, 6
Prime numbers: 2, 3, 5 (three outcomes)
P(prime) = 3/6 = ½
```
**Answer**: **½**

---

### Problem 3 (MEDIUM — Quadratic Factorisation)
**Q**: Solve x² − 7x + 10 = 0

**Solution**:
```
Find two numbers: product = +10, sum = −7 → (−5) and (−2)
(x − 5)(x − 2) = 0
x = 5  or  x = 2
```
**Answer**: **x = 5 or x = 2**

---

### Problem 4 (MEDIUM — Volume)
**Q**: Find the volume of a cone with radius 6 cm and height 14 cm. (π = 22/7)

**Solution**:
```
V = ⅓ × π × r² × h
  = ⅓ × 22/7 × 36 × 14
  = ⅓ × 22 × 36 × 2          (14/7 = 2)
  = ⅓ × 1,584
  = 528 cm³
```
**Answer**: **528 cm³**

---

### Problem 5 (HARD — Inverse Variation Word Problem)
**Q**: The time (t hours) needed to build a fence varies inversely as the number of workers (n). If 6 workers build the fence in 4 hours, how long will 8 workers take?

**Solution**:
```
t ∝ 1/n → t = k/n
k = tn = 4 × 6 = 24
When n = 8: t = 24/8 = 3 hours
```
**Answer**: **3 hours**

## 3. COMMON MISTAKES IN JSS3 MATHEMATICS

| Mistake | Correction |
|---------|-----------|
| Using c² = a² − b² for Pythagoras | Correct formula: **c² = a² + b²** (or a² = c² − b² for a leg) |
| Forgetting ± in square root of quadratic | Both + and − roots must be stated |
| P(event) > 1 | Probability is always between 0 and 1; recheck calculation |
| Volume using SA formula | Volume and Surface Area have different formulas; know both for each solid |
| Direct variation: y ÷ x instead of y = kx | Find k first; then use y = kx for new values |

## 5. EXAM QUICK TIPS

**Tip 1**: **Pythagoras** — identify the hypotenuse first (always opposite the right angle)

**Tip 2**: **Quadratic factorisation** — ask "two numbers that MULTIPLY to c and ADD to b"

**Tip 3**: **Probability** — count total outcomes carefully (especially "and"/"or" events)

**Tip 4**: **Volume/Area** — write the formula, substitute, then simplify step by step

**Tip 5**: **Variation** — always find k first; then apply the formula for the unknown
