## 1. KEY CONCEPTS (Master These)

### Concept 1: Types of Algebraic Equations

**Linear Equations** (Most common in JAMB)
- Form: ax + b = 0
- Example: 2x + 5 = 13
- Solution: x = 4
- Complexity: Basic (40% of algebra questions)

**Quadratic Equations**
- Form: ax² + bx + c = 0
- Example: x² + 5x + 6 = 0
- Solution: x = -2 or x = -3 (using factorization)
- Complexity: Medium (40% of algebra questions)

**Simultaneous Equations**
- Form: Two equations with two unknowns
- Example: 2x + y = 10; x - y = 1
- Solution: x = 3, y = 4
- Complexity: Hard (15% of algebra questions)

**Fractional Equations**
- Form: Equations with fractions
- Example: (x+2)/3 = 5
- Solution: x = 13
- Complexity: Medium (5% of algebra questions)

---

### Concept 2: Rules for Solving Equations

**Rule 1: Do Same Thing to Both Sides**
```
Original: 2x + 5 = 13
Subtract 5 from both sides: 2x = 8
Divide by 2: x = 4
```

**Rule 2: BODMAS/PEMDAS Order**
- Brackets/Parentheses
- Orders/Exponents
- Division & Multiplication (left to right)
- Addition & Subtraction (left to right)

**Rule 3: Collecting Like Terms**
```
3x + 2x + 5 - 3 = 9
5x + 2 = 9
5x = 7
x = 7/5 = 1.4
```

**Rule 4: Factorization**
```
x² + 5x + 6 = 0
(x + 2)(x + 3) = 0
x = -2 or x = -3
```

---

### Concept 3: Common Algebraic Mistakes

| Mistake | Example | Correct |
|---------|---------|---------|
| Forgetting to multiply/divide both sides | 2x = 10 → x = 5 ✓ BUT forgetting to divide by 2 | Always divide/multiply BOTH sides |
| Sign errors | -3x = 12 → x = 4 ✗ (Should be x = -4) | When dividing by negative, flip sign |
| Incorrect factorization | x² + 5x + 6 = (x+2)(x+2) ✗ | Check: (x+2)(x+3) is correct |
| Ignoring negative solutions | x² = 4 → x = 2 ✓ BUT also x = -2 | Quadratic equations have TWO solutions |
| Order of operations | 2 + 3 × 4 = 20 ✗ (Should be 14) | Multiply before adding |

## 2. SAMPLE PROBLEMS (5 Increasing Difficulty)

### Problem 1 (EASY - Linear)
**Q**: Solve for x: 3x + 2 = 11

**Options**:
A) 1  
B) 2  
C) 3 ← **CORRECT**  
D) 4  

**Solution**:
```
3x + 2 = 11
3x = 11 - 2  (subtract 2 from both sides)
3x = 9
x = 3  (divide by 3)
```

**Check**: 3(3) + 2 = 9 + 2 = 11 ✓

**Key Concept**: Isolate x by doing opposite operations

---

### Problem 2 (EASY - Basic Algebra)
**Q**: Simplify: 5x + 3x - 2x

**Options**:
A) 6x  
B) 8x  
C) 6x ← **CORRECT**  
D) 10x  

**Solution**:
```
5x + 3x - 2x
= (5 + 3 - 2)x
= 6x
```

**Key Concept**: Combine like terms (only x's with x's)

---

### Problem 3 (MEDIUM - Quadratic)
**Q**: Solve: x² + 7x + 12 = 0

**Options**:
A) x = -3, -4 ← **CORRECT**  
B) x = 3, 4  
C) x = -3, 4  
D) x = 3, -4  

**Solution**:
```
x² + 7x + 12 = 0

Factor: Find two numbers that multiply to 12 and add to 7
Numbers: 3 and 4 (3 × 4 = 12, 3 + 4 = 7)

(x + 3)(x + 4) = 0

Solutions: x = -3 or x = -4
```

**Check**: 
- If x = -3: (-3)² + 7(-3) + 12 = 9 - 21 + 12 = 0 ✓
- If x = -4: (-4)² + 7(-4) + 12 = 16 - 28 + 12 = 0 ✓

**Key Concept**: Quadratic equations have TWO solutions

---

### Problem 4 (MEDIUM - Fractional)
**Q**: Solve: (2x + 3)/5 = 7

**Options**:
A) 16  
B) 14  
C) 15  
D) 17 ← **CORRECT**  

**Solution**:
```
(2x + 3)/5 = 7

Multiply both sides by 5:
2x + 3 = 35

Subtract 3:
2x = 32

Divide by 2:
x = 16... wait

Let me recalculate:
(2x + 3)/5 = 7
2x + 3 = 35
2x = 32
x = 16

Check: (2(16) + 3)/5 = (32 + 3)/5 = 35/5 = 7 ✓
```

Wait, let me verify the options. If x = 16: (32+3)/5 = 35/5 = 7 ✓

The answer should be A (16), but showing as D in the options. Let me make a different problem.

**REVISED Problem 4**:
**Q**: Solve: (3x - 2)/4 = 5

**Solution**:
```
Multiply both sides by 4:
3x - 2 = 20

Add 2:
3x = 22

Divide by 3:
x = 22/3 ≈ 7.33
```

**Key Concept**: Clear fractions by multiplying both sides by denominator

### Problem 5 (HARD - Simultaneous)
**Q**: Solve the system:
```
2x + y = 10
x - y = 2
```

**Options**:
A) x = 4, y = 2 ← **CORRECT**  
B) x = 2, y = 4  
C) x = 3, y = 3  
D) x = 5, y = 0  

**Solution - Method 1 (Elimination)**:
```
Equation 1: 2x + y = 10
Equation 2: x - y = 2

Add the equations (y cancels out):
3x = 12
x = 4

Substitute back into Equation 2:
4 - y = 2
y = 2
```

**Check**:
- Equation 1: 2(4) + 2 = 8 + 2 = 10 ✓
- Equation 2: 4 - 2 = 2 ✓

**Key Concept**: Simultaneous equations = solve by elimination or substitution

## 3. COMMON MISTAKES IN ALGEBRA

| Mistake | Example | Why Wrong | Fix |
|---------|---------|-----------|-----|
| Sign error | -2x = 6 → x = 3 | Should be x = -3 | When dividing by negative, flip sign |
| Forgetting to apply to both sides | 2x + 5 = 13 → 2x = 8 (forgot -5) | Only subtracted from one side | Do SAME operation on both sides |
| Incorrect factorization | x² + 5x + 4 = (x+1)(x+5) | (1)(5) = 5, (1+5) = 6, not 5 | (x+1)(x+4): (1)(4)=4, (1+4)=5 ✓ |
| Wrong order of operations | 2 + 3 × 2 = 10 | Multiplied first then added → 6+4=10 | Multiply FIRST: 2 + 6 = 8 |
| Not checking solutions | x² = 4 → x = 2 only | Forgot x = -2 | Always check both positive & negative |

## 4. EXAM QUICK TIPS

**Tip 1: Isolate the variable**
- Get x on one side, numbers on other
- Use inverse operations (+ becomes -, × becomes ÷)

**Tip 2: Check your answer**
- Plug x back into original equation
- If it works, you're correct

**Tip 3: Watch your signs**
- Negative × negative = positive
- Negative ÷ negative = positive
- Many students lose marks here

**Tip 4: For quadratics, find two solutions**
- Factorization gives (x + a)(x + b) = 0
- Solutions are x = -a and x = -b

**Tip 5: Time management**
- Easy equations (30 sec each)
- Medium equations (1-2 min each)
- Hard equations (2-3 min each)

## 5. WHAT TO MEMORIZE

**MUST MEMORIZE**:
```
BODMAS: Brackets, Orders, Division/Multiplication, Addition/Subtraction
Factorization pattern: x² + (a+b)x + ab = (x+a)(x+b)
Quadratic formula: x = [-b ± √(b²-4ac)] / 2a (for hard problems)
```

**Common Factors to Know**:
```
x² - 4 = (x-2)(x+2)  [difference of squares]
x² + 2x + 1 = (x+1)²  [perfect square]
x² - 6x + 9 = (x-3)²  [perfect square]
```
