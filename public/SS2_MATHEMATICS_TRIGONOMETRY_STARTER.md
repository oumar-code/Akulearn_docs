## 1. KEY CONCEPTS (Master These)

### Concept 1: Trigonometric Ratios (SOHCAHTOA)

**The 3 Basic Ratios** (for a right-angled triangle):
```
          hypotenuse
        /|
       / |
      /  | opposite
     /   |
    /θ   |
   /_____|
   adjacent
```

**SOHCAHTOA**:
```
SOH → Sin θ = Opposite / Hypotenuse
CAH → Cos θ = Adjacent / Hypotenuse
TOA → Tan θ = Opposite / Adjacent
```

**Reciprocal Ratios**:
```
Cosecant:  cosec θ = 1/sin θ = Hypotenuse / Opposite
Secant:    sec θ   = 1/cos θ = Hypotenuse / Adjacent
Cotangent: cot θ   = 1/tan θ = Adjacent / Opposite
```

**Why Important**: 6-8 WAEC/JAMB questions per exam use trig ratios

---

### Concept 2: Special Angles (Must Memorize)

| Angle | Sin | Cos | Tan |
|-------|-----|-----|-----|
| 0° | 0 | 1 | 0 |
| 30° | ½ | √3/2 | 1/√3 |
| 45° | √2/2 | √2/2 | 1 |
| 60° | √3/2 | ½ | √3 |
| 90° | 1 | 0 | undefined |

**Memory Trick for Sin**:
```
sin 0° = √0/2 = 0
sin 30° = √1/2 = ½
sin 45° = √2/2
sin 60° = √3/2
sin 90° = √4/2 = 1
```
Pattern: sin increases 0 → 1 as angle increases 0° → 90°

**Memory Trick for Cos**: Cos values are Sin values in REVERSE order

---

### Concept 3: Angles in All Four Quadrants (CAST Rule)

```
         90°
          |
 (II)     |    (I)
S only    |  ALL positive
 Sin+     |  Sin+, Cos+, Tan+
          |
180°------+------0°/360°
          |
 (III)    |   (IV)
Tan only  | Cos only
 Tan+     |  Cos+
          |
         270°
```

**CAST Rule** (starting from quadrant IV, going anti-clockwise):
- **C**os positive in IV
- **A**ll positive in I
- **S**in positive in II
- **T**an positive in III

**Memory**: "**C**ast" or "**A**ll **S**tudents **T**ake **C**alculus"

---

### Concept 4: Trigonometric Identities

**Fundamental Identities** (appear directly in exams):
```
sin²θ + cos²θ = 1      (Pythagoras identity)
tan θ = sin θ / cos θ
1 + tan²θ = sec²θ
1 + cot²θ = cosec²θ
```

**Useful Forms**:
```
sin²θ = 1 - cos²θ
cos²θ = 1 - sin²θ
tan²θ = sec²θ - 1
```

**Exam Use**: Simplify expressions or prove identities using these

## 2. SAMPLE PROBLEMS (5 Increasing Difficulty)

### Problem 1 (EASY – Basic Ratio)
**Q**: In a right-angled triangle, if the opposite side is 3 and the hypotenuse is 5, find sin θ.

**Options**:
A) 4/5
B) 3/4
C) 3/5 ← **CORRECT**
D) 5/3

**Solution**:
```
sin θ = Opposite / Hypotenuse = 3 / 5

Using Pythagoras: adjacent = √(5² - 3²) = √(25-9) = √16 = 4
cos θ = 4/5 and tan θ = 3/4
```

**Key Concept**: SOH → sin = Opposite / Hypotenuse

---

### Problem 2 (EASY – Special Angles)
**Q**: Find the exact value of: tan 60°

**Options**:
A) 1/√3
B) √3/2
C) 1
D) √3 ← **CORRECT**

**Solution**:
```
From the special angles table:
tan 60° = √3

Quick check: sin 60° / cos 60° = (√3/2) / (½) = √3 ✓
```

**Key Concept**: Memorize special angles (30°, 45°, 60°)

---

### Problem 3 (MEDIUM – CAST Rule)
**Q**: If sin θ = 3/5 and θ is in the second quadrant, find cos θ.

**Options**:
A) 4/5
B) –4/5 ← **CORRECT**
C) 3/4
D) –3/4

**Solution**:
```
Step 1: Find the magnitude of cos θ
sin²θ + cos²θ = 1
(3/5)² + cos²θ = 1
9/25 + cos²θ = 1
cos²θ = 16/25
|cos θ| = 4/5

Step 2: Apply CAST rule
Second quadrant → Sin positive, Cos NEGATIVE
∴ cos θ = –4/5
```

**Key Concept**: CAST tells you the SIGN; Pythagoras identity gives the MAGNITUDE

---

### Problem 4 (MEDIUM – Identity Application)
**Q**: Simplify: (sin²θ + cos²θ) / cos²θ

**Options**:
A) sin²θ
B) 1
C) sec²θ ← **CORRECT**
D) tan²θ

**Solution**:
```
(sin²θ + cos²θ) / cos²θ

= sin²θ/cos²θ + cos²θ/cos²θ   (split the fraction)

= tan²θ + 1

= sec²θ    (using identity: 1 + tan²θ = sec²θ)
```

**Key Concept**: Know identities and how to apply them in simplification

---

### Problem 5 (HARD – Equation Solving)
**Q**: Solve for θ in the range 0° ≤ θ ≤ 360°: 2sin²θ – sinθ – 1 = 0

**Options**:
A) 30°, 150°
B) 90°, 210°, 330°
C) 90°, 210°, 330° ← **CORRECT**
D) 30°, 90°, 270°

**Solution**:
```
Let u = sin θ:
2u² – u – 1 = 0

Factorize:
(2u + 1)(u – 1) = 0

u = –½  or  u = 1

Case 1: sin θ = 1
θ = 90°

Case 2: sin θ = –½
Reference angle = 30°
Sin is negative in quadrants III and IV
θ = 180° + 30° = 210°
θ = 360° – 30° = 330°

Solutions: θ = 90°, 210°, 330°
```

**Key Concept**: Treat trig equations like quadratic equations, then apply CAST for signs

## 3. COMMON MISTAKES IN TRIGONOMETRY

| Mistake | Example | Correct Approach |
|---------|---------|-----------------|
| Wrong SOHCAHTOA application | sin θ = adjacent/hypotenuse | sin = Opposite / Hypotenuse |
| Forgetting signs in quadrants | sin 210° = +½ | CAST: III quadrant → sin negative → sin 210° = –½ |
| Wrong special angle values | tan 45° = √3 | tan 45° = 1 (not √3, that's tan 60°) |
| Not checking range when solving | Only find one solution | Always find ALL solutions in given range |
| Using degrees in calculator set to radians | Wrong numerical answer | Match calculator mode to angle unit used |

## 5. EXAM QUICK TIPS

**Tip 1**: **SOHCAHTOA first** – draw and label the triangle before applying formulas

**Tip 2**: **Memorize special angles** – at least 30°, 45°, 60° appear in almost every exam

**Tip 3**: **CAST for signs** – after finding magnitude using Pythagoras, apply CAST for correct sign

**Tip 4**: **Trig equations = quadratic technique**
- Substitute u = sin θ (or cos θ)
- Solve like a quadratic
- Find all angles in the given range

**Tip 5**: **Check identity questions with substitution**
- If stuck proving an identity, substitute θ = 45° to check if both sides equal → helps confirm direction
