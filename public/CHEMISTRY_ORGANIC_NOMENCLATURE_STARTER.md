## 1. KEY CONCEPTS (To be enriched by expert)

### Concept 1: IUPAC Naming System
**Definition**: International system for naming organic compounds systematically
**Why Important**: 5-8 JAMB questions per exam focus on naming
**Exam Weight**: HIGH

**Components**:
- Functional group identification
- Carbon chain counting
- Position numbering
- Substituent naming

**Real Exam Questions**:
- "What is the IUPAC name of CH3-CH2-OH?"
- "Name the compound: CH3-CH(OH)-CH3"
- "Identify the correct name: CH3-COOH"

---

### Concept 2: Functional Group Priorities
**Definition**: Rules for which functional group determines compound class
**Priority Order** (1=highest):
1. Carboxylic acid (-COOH)
2. Aldehyde (-CHO)
3. Ketone (C=O)
4. Alcohol (-OH)
5. Alkene (C=C)
6. Alkyne (C≡C)

**Exam Tip**: If multiple functional groups, pick highest priority for name

---

### Concept 3: Carbon Chain Prefixes
**These MUST be memorized**:
- 1 carbon = Meth-
- 2 carbons = Eth-
- 3 carbons = Prop-
- 4 carbons = But-
- 5 carbons = Pent-
- 6 carbons = Hex-
- 7 carbons = Hept-
- 8 carbons = Oct-

**Quick Drill**: Close your eyes. Can you name 1-8 carbons? Practice until automatic!

## 2. PRACTICE QUESTIONS (5 Increasing Difficulty)

### Question 1 (EASY) - 40% of exam questions
**Q**: What is the IUPAC name of this compound? CH₃-CH₂-OH

**Options**:
A) Methanol  
B) Ethanol ← **CORRECT**  
C) Propanol  
D) Butanol  

**Explanation**: 
- Count carbons: C-C = 2 carbons = "Eth-"
- Functional group: -OH = "ol"
- Name: Ethanol
- Common name: Ethyl alcohol

**Key Learning**: Base name = prefix (carbon count) + suffix (functional group)

---

### Question 2 (EASY)
**Q**: How many carbons are in a molecule named "Butane"?

**Options**:
A) 2  
B) 3  
C) 4 ← **CORRECT**  
D) 5  

**Explanation**: 
- "But-" prefix = 4 carbons
- Learn: Meth(1), Eth(2), Prop(3), But(4)

---

### Question 3 (MEDIUM) - 50% of exam questions
**Q**: What is the IUPAC name of this compound?
```
    CH₃
    |
CH₃-CH-CH₃
```

**Options**:
A) 1-Propanol  
B) 2-Propanol ← **CORRECT**  
C) Propane  
D) 3-Propanol  

**Explanation**:
- Total carbons: 3 = "Prop-"
- Central carbon has -OH group (alcohol)
- Number from end closest to -OH: positions are 1, 2, 3
- -OH is at position 2
- Name: 2-Propanol (or isopropanol)

**Common Mistake**: Students number from left (wrong!). Always number from nearest functional group.

---

### Question 4 (MEDIUM)
**Q**: What is the name of: CH₃-COOH

**Options**:
A) Methanoic acid ← **CORRECT**  
B) Ethanoic acid  
C) Propanoic acid  
D) Formic acid  

**Explanation**:
- Functional group: -COOH = carboxylic acid
- Carbon count: 1 carbon (the C in COOH counts) = "meth-"
- Name: Methanoic acid (also called formic acid)

**Key Concept**: COOH counts as ONE carbon for naming purposes!

---

### Question 5 (HARD) - 25% of exam questions
**Q**: What is the IUPAC name of:
```
    CH₃
    |
CH₃-CH-CH₂-COOH
```

**Options**:
A) 3-Methylbutanoic acid  
B) 2-Methylpropanoic acid ← **CORRECT**  
C) 3-Methylpropanoic acid  
D) 2-Methylbutanoic acid  

**Explanation**:
1. Find functional group: -COOH (carboxylic acid)
2. Count main chain including COOH: 3 carbons = "Prop-"
3. Add suffix: "propanoic acid"
4. Find substituents: CH₃ (methyl) branch
5. Number from COOH: COOH=1, next=2, branch on position 2
6. Name: 2-Methylpropanoic acid

**Why (A) is wrong**: If you count 4 carbons, you'd get butanoic acid. Always count only the carbons in the main chain!

## 3. COMMON EXAM MISTAKES

| Mistake | Why Wrong | Correct Approach |
|---------|-----------|-------------------|
| Numbering from wrong end | Functional group must get LOWEST number | Always start from group -OH, -COOH, etc. |
| Forgetting the "E" before "OIC" | Grammar rule in chemistry | Prop → Propanoic (NOT Propnoic) |
| Counting COOH as 2 carbons | It's ONE carbon for naming | CH₃-COOH = Eth-anoic (2C total) |
| Missing position numbers | Position matters! | 2-Propanol (NOT propanol) |
| Wrong functional group | Determines entire name | Always ID group first! |

---

## 4. EXAM QUICK TIPS

**Tip 1**: Draw the structure → Circle the functional group → Apply rules

**Tip 2**: Memorize the 8 prefixes (Meth-Eth-Prop-But-Pent-Hex-Hept-Oct). These appear in almost EVERY exam.

**Tip 3**: When multiple functional groups exist, pick the one with HIGHEST priority (COOH > CHO > C=O > OH > C=C)

**Tip 4**: The position number goes BETWEEN the prefix and the suffix
- ✅ Correct: 2-Methylpropanoic acid
- ❌ Wrong: Methylpropanoic acid or 2-Propanoic acid

**Tip 5**: If unsure, draw → count → number → name. Takes 30 seconds, prevents mistakes.

## 5. WHAT TO MEMORIZE

**MUST MEMORIZE** (appears in 80% of questions):
```
Carbon prefixes (1-8):
Meth=1, Eth=2, Prop=3, But=4, Pent=5, Hex=6, Hept=7, Oct=8

Functional group suffixes:
-OH = alcohol (-ol)
-COOH = carboxylic acid (-oic acid)
-CHO = aldehyde (-al)
-C=O = ketone (-one)
-C=C = alkene (-ene)
-C≡C = alkyne (-yne)

Functional group priorities (for naming):
COOH > CHO > C=O > OH > Alkene > Alkyne
```
