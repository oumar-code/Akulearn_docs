## 1. KEY CONCEPTS (Master These)

### Concept 1: What are Hydrocarbons?

**Definition**: Organic compounds containing ONLY hydrogen and carbon atoms

**Three Main Classes**:

| Class | General Formula | Bond Type | Example |
|-------|----------------|-----------|---------|
| Alkanes | CₙH₂ₙ₊₂ | Single bond (C–C) | Methane (CH₄) |
| Alkenes | CₙH₂ₙ | One double bond (C=C) | Ethene (C₂H₄) |
| Alkynes | CₙH₂ₙ₋₂ | One triple bond (C≡C) | Ethyne (C₂H₂) |

**Why Important**: 5-8 JAMB questions per exam on structures, naming, and reactions

---

### Concept 2: Alkanes (Saturated Hydrocarbons)

**Definition**: Hydrocarbons with only single bonds (no double or triple bonds); described as **saturated**

**First 6 Alkanes** (Must Memorize):
```
1 C → Methane    (CH₄)     → Natural gas, cooking fuel
2 C → Ethane     (C₂H₆)    → Component of natural gas
3 C → Propane    (C₃H₈)    → LPG (cooking gas)
4 C → Butane     (C₄H₁₀)   → Lighter fuel, LPG
5 C → Pentane    (C₅H₁₂)   → Gasoline component
6 C → Hexane     (C₆H₁₄)   → Solvent
```

**Key Properties of Alkanes**:
- Undergo **substitution reactions** (not addition)
- Burn in oxygen to give CO₂ and H₂O (combustion)
- Relatively unreactive (no double/triple bonds to attack)

**Combustion Reaction (Important!)**:
```
CH₄ + 2O₂ → CO₂ + 2H₂O  (complete combustion)
2CH₄ + 3O₂ → 2CO + 4H₂O  (incomplete combustion)
```

---

### Concept 3: Alkenes (Unsaturated Hydrocarbons)

**Definition**: Hydrocarbons with at least ONE C=C double bond; described as **unsaturated**

**First 4 Alkenes**:
```
2 C → Ethene    (C₂H₄)    → Making plastics (polyethylene)
3 C → Propene   (C₃H₆)    → Making polypropylene
4 C → But-1-ene (C₄H₈)    → Synthetic rubber
5 C → Pent-1-ene (C₅H₁₀)  → Chemical synthesis
```

**Key Reactions of Alkenes** (Highly Tested):
```
1. Addition of H₂ (Hydrogenation):
   CH₂=CH₂ + H₂ → CH₃-CH₃  (ethene → ethane)

2. Addition of Br₂ (Bromine water test):
   CH₂=CH₂ + Br₂ → CH₂Br-CH₂Br  (decolourises brown bromine water)

3. Addition of HBr (Markovnikov):
   CH₃-CH=CH₂ + HBr → CH₃-CHBr-CH₃  (major product)

4. Polymerisation:
   n(CH₂=CH₂) → -(CH₂-CH₂)ₙ-  (polythene/polyethylene)
```

**Bromine Water Test**: Brown/orange → colourless = alkene (unsaturated) confirmed!

---

### Concept 4: Alkynes (Unsaturated Hydrocarbons)

**Definition**: Hydrocarbons with at least ONE C≡C triple bond

**Most Important Alkyne**:
```
Ethyne (Acetylene): C₂H₂   (HC≡CH)
- Used in welding torches (oxyacetylene flame)
- Made by reacting calcium carbide with water:
  CaC₂ + 2H₂O → Ca(OH)₂ + C₂H₂↑
```

**General Properties of Alkynes**:
- More unsaturated than alkenes → more reactive
- Undergo addition reactions (like alkenes)
- Can add 2 moles of H₂ (or Br₂) due to triple bond

---

### Concept 5: Distinguishing Between Alkanes, Alkenes, Alkynes

| Test | Alkane | Alkene | Alkyne |
|------|--------|--------|--------|
| Bromine water (Br₂) | No change (stays brown) | Decolourises | Decolourises |
| KMnO₄ solution | No change (stays purple) | Decolourises | Decolourises |
| H₂ addition | No reaction | Adds 1 mol H₂ | Adds 2 mol H₂ |
| Combustion | Burns (smokeless) | Burns (smoky) | Burns (very smoky) |

**Exam Favourite**: "Which of these will decolourise bromine water?"
- Answer: Alkene or Alkyne (unsaturated compounds)

## 2. SAMPLE PROBLEMS (5 Increasing Difficulty)

### Problem 1 (EASY – Identification)
**Q**: What is the general formula for alkanes?

**Options**:
A) CₙH₂ₙ
B) CₙH₂ₙ₋₂
C) CₙH₂ₙ₊₂ ← **CORRECT**
D) CₙH₂ₙ₊₁

**Explanation**:
- Alkanes: CₙH₂ₙ₊₂ (saturated, single bonds)
- Alkenes: CₙH₂ₙ (one double bond)
- Alkynes: CₙH₂ₙ₋₂ (one triple bond)
- Memory: As degree of unsaturation increases, H count decreases

**Key Concept**: Alkanes +2, Alkenes no adjustment, Alkynes –2

---

### Problem 2 (EASY – Naming)
**Q**: What is the molecular formula of propane?

**Options**:
A) C₂H₆
B) C₃H₈ ← **CORRECT**
C) C₃H₆
D) C₄H₁₀

**Solution**:
```
Propane = 3 carbons (prop- prefix)
Alkane formula: CₙH₂ₙ₊₂
n = 3 → C₃H₂(3)+2 = C₃H₈
```

**Key Concept**: Apply alkane formula CₙH₂ₙ₊₂ with n = number of carbons

---

### Problem 3 (MEDIUM – Reactions)
**Q**: What observation is made when ethene gas is bubbled through bromine water?

**Options**:
A) Brown bromine water turns colourless ← **CORRECT**
B) Bromine water turns blue
C) No observable change
D) White precipitate forms

**Explanation**:
```
CH₂=CH₂ + Br₂ → BrCH₂-CH₂Br
(ethene)  (brown)  (colourless)

The Br₂ adds across the double bond
The brown colour of Br₂ disappears → water turns colourless
This confirms the compound is unsaturated (alkene or alkyne)
```

**Key Concept**: Decolourisation of bromine water = unsaturated hydrocarbon

---

### Problem 4 (MEDIUM – Combustion Calculation)
**Q**: What are the products of the complete combustion of butane (C₄H₁₀)?

**Options**:
A) CO and H₂O only
B) C and H₂O
C) CO₂ and H₂O ← **CORRECT**
D) CO₂ and H₂ only

**Solution**:
```
Complete combustion equation for butane:
2C₄H₁₀ + 13O₂ → 8CO₂ + 10H₂O

Complete combustion always gives:
- Carbon dioxide (CO₂)
- Water (H₂O)

Incomplete combustion (limited O₂) gives:
- Carbon monoxide (CO) or carbon (soot)
```

**Key Concept**: Complete combustion → CO₂ + H₂O; Incomplete → CO or C

---

### Problem 5 (HARD – Isomers & Structure)
**Q**: How many structural isomers does butane (C₄H₁₀) have?

**Options**:
A) 1
B) 2 ← **CORRECT**
C) 3
D) 4

**Explanation**:
```
C₄H₁₀ has 2 structural isomers:

Isomer 1: n-butane (straight chain)
CH₃-CH₂-CH₂-CH₃

Isomer 2: Isobutane / 2-methylpropane (branched)
    CH₃
     |
CH₃-CH-CH₃

Both have formula C₄H₁₀ but different arrangements!
Pentane (C₅H₁₂) has 3 isomers; hexane (C₆H₁₄) has 5 isomers
```

**Key Concept**: Structural isomers = same molecular formula, different structural arrangement

## 3. COMMON MISTAKES IN HYDROCARBONS

| Mistake | Why Wrong | Correct Understanding |
|---------|-----------|------------------------|
| Confusing alkane and alkene formulas | Mixing up CₙH₂ₙ₊₂ and CₙH₂ₙ | Alkane has +2 H; alkene has no adjustment |
| Thinking alkanes decolourise bromine water | Alkanes are saturated (unreactive) | Only unsaturated (alkenes/alkynes) decolourise |
| Wrong combustion products | Writing O₂ instead of CO₂ | Complete combustion always gives CO₂ + H₂O |
| Forgetting Markovnikov's rule | Getting wrong product in HBr addition | H adds to C with more H; Br to C with fewer H |
| Missing isomers | Only drawing straight chain | Always check for branched chain isomers |

```
ALKANES (saturated)
 CH₄  C₂H₆  C₃H₈  C₄H₁₀ ...
 |    |      |      |

ALKENES (one double bond)
```

```
Alkane → Bromine water stays BROWN (no reaction)
Alkene → Bromine water turns COLOURLESS (addition reaction)

```

```

```

## 5. EXAM QUICK TIPS

**Tip 1**: **General formula differentiation**
- Alkanes CₙH₂ₙ₊₂, Alkenes CₙH₂ₙ, Alkynes CₙH₂ₙ₋₂
- Each step = 2 fewer H atoms

**Tip 2**: **Bromine water test = key practical**
- Unsaturated (alkene/alkyne) → decolourises
- Saturated (alkane) → no change

**Tip 3**: **Combustion products**
- Complete: CO₂ + H₂O (always)
- Incomplete: CO + H₂O (limited oxygen)

**Tip 4**: **Name from structure**
- Count carbons in longest chain → get prefix (Meth/Eth/Prop/But/Pent/Hex)
- Identify bond type → -ane/-ene/-yne suffix
- Add position number for double/triple bonds: But-1-ene

**Tip 5**: **Alkene reactions to know**
- + H₂ → alkane (hydrogenation)
- + Br₂ → dibromo compound (halogenation, decolourisation test)
- + H₂O → alcohol (hydration)
- n × monomer → polymer (polymerisation)
