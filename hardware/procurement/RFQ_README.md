# RFQ Pack — Edge Alpha PMU

This folder contains materials to request quotes for the Edge Alpha PMU prototype.

Contents
- `edge_alpha_rfq_filled.csv` — populated RFQ/BOM (prototype quantities)
- `pmu_schematic.pdf` — schematic textual export (for supplier reference)
- `rfq_pack_edge_alpha.zip` — zipped bundle for suppliers
- `email_requests/` — per-supplier email drafts (digikey, mouser, arrow, etc.)

How to use
1. Review `edge_alpha_rfq_filled.csv` and adjust quantities as needed.
2. Attach `pmu_schematic.pdf` and the CSV to supplier emails (templates are in `email_requests/`).
3. Request lead times, MOQs, alternate parts, and pricing for prototype quantities.

Notes
- The BOM with manufacturer part numbers is in `hardware/boms/edge_alpha_bom_with_mfg_pn.csv` for procurement use.
- For high-confidence quotes, include footprint/PCB PDF exports and any required test/assembly notes.
