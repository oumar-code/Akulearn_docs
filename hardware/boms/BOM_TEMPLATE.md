# BOM Template

Use this template to create per-project BOMs. Include manufacturer, vendor SKU, alternate SKU, unit price, quantity, lead time, and notes about certifications.

Columns:
- Part Name
- Manufacturer
- Manufacturer Part Number (MPN)
- Supplier / Distributor
- Supplier SKU
- Alternate Supplier
- Unit Price (USD)
- Quantity
- Lead Time (weeks)
- RoHS / Regulatory Notes
