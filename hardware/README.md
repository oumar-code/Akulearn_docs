# Aku Platform - Hardware Design folder

This folder will house hardware schematics, BOMs, CAD models, and design notes for the Aku Platform tiers (Edge, Super Hub, IG-Hub).

Structure:
- `edge/` - Edge Hub schematics, KiCad project, BOM, enclosure CAD
- `superhub/` - Super Hub integration notes, rack diagrams, BOM
- `ig-hub/` - IG-Hub co-location designs, network/cabling diagrams
- `boms/` - Cross-tier BOM templates and supplier lists
- `schematics/` - shared schematic fragments and symbols

See the hardware strategy document for the phased plan and decision criteria.
