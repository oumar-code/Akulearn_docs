## Technical Specifications

### File Format Requirements
- **Primary Format:** GLB (binary glTF 2.0)
- **Fallback Formats:** OBJ, FBX (for Unity integration)
- **File Size Target:** < 5MB per model (optimized for mobile)
- **Texture Resolution:** 1024x1024 max (mobile-optimized)
- **Polygon Count:** < 50,000 triangles per model
